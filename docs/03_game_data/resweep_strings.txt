=== 19. BONUS: HIGH ENTROPY REGION ANALYSIS ===

Offset 0x000F8000-0x0010A000 (~168KB) contains HIGH ENTROPY data (7.1-7.3
bits/byte). This region is where the majority of the garbled/obfuscated
printf-style strings were extracted. These appear to be:
  (a) XOR-encrypted or scrambled format strings
  (b) Possibly a custom string table with runtime deobfuscation
  (c) Could be compressed data decoded at runtime

The sheer volume (~2600+ garbled strings in the 0x0F8000 region) suggests
a systematic string obfuscation approach for anti-analysis.
