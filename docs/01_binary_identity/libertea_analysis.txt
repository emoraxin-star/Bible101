--- SC FARMING LOOP MECHANICS ---

  1. Probe a mission to capture data (entityDeep, entityDataDeep, warTime)
  2. Store capture in C:\libertea_replay_cap.json
  3. Auto-Replay fires batches of 9 API calls to:
     POST api.live.prod.thehelldiversgame.com/api/Operation/Mission/end
  4. Each batch: 9 calls x 500ms apart
  5. Cooldown between batches: 58 seconds
  6. Alternates SC and Medal batches
  7. Crash recovery via Vectored Exception Handler (VEH)
  8. SC goal auto-stops when reached
  9. Auto-sync distributes SC across lobby players
  10. NOPs hash table INSERT calls to prevent duplicate detection

--- INJECTOR BYPASS TECHNIQUE ---

  1. Enable SeDebugPrivilege via ntdll.dll
  2. Target: helldivers2.exe
  3. Block/remove GameMon.des / GameMon64.des (GameGuard drivers)
  4. Inject LIBERTEA.DLL via CreateRemoteThread or similar
  5. DLL loads, DllMain decompresses .text via custom aPLib variant
  6. Resolve imports, scan for game.dll patterns, install hooks
  7. Create ImGui overlay window (LIBERTEAWnd class)
  8. Hook wglSwapIntervalEXT / Present for overlay rendering
